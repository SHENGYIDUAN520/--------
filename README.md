# 智能陪护监控平台开发进度

## 项目简介
智能陪护监控平台是一个用于老年人健康监控和远程照护的系统。平台通过智能设备收集老年人的生命体征数据，实现远程监控、异常预警和紧急救助等功能。

## 开发环境
- 前端：HTML5, CSS3, JavaScript
- UI库：Bootstrap Icons
- 后端：暂未实现

## 功能模块
- 仪表盘（Dashboard）：展示总体监控数据
- 老人详情（Detail）：查看单个老人的详细数据
- 历史数据（History）：查询历史数据记录
- 系统设置（Settings）：配置系统参数
